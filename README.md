# Dart Counter PWA

A modern Progressive Web App for counting darts scores across all devices.  
Designed to be fast, simple, and accessible with a responsive interface for desktop and mobile use.

## Features
- Cross-device support
- Fast and responsive design
- PWA support
- Simple darts score tracking

## Installation
[Installation guide](PWA_INSTALLATION_GUIDE.md)
(for the "node_modules" folder use "npm install" command to download everything)
